# 📱 LumiaSend
**Bridge the gap between your PC and your Windows Phone.**

LumiaSend is a lightweight local server designed to make transferring text, links, and files to legacy Windows Phone 8.1 and Windows 10 Mobile devices effortless.



## 🚀 Features
* **Instant Sharing:** Send URLs or snippets to your phone's clipboard.
* **File Transfers:** Push photos and .appx installers directly to your device.
* **No Cloud Required:** Everything stays on your local Wi-Fi.

## 📥 Installation
1. Download this repo and extract it.
2. Run `Install_Requirements.bat` to install Python dependencies.
3. Run `receiver.exe` or `receiver.py` (or your compiled .exe launcher).
4. Enter the IP address shown on your PC into the Lumia app.
